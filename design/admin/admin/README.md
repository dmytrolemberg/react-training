# Minimal Shop Admin Portal

Separate responsive static admin portal for managing the minimal online shop.

## Design

Chosen style: **Variant 3 — Airy SaaS structure**, adapted to the same warm minimal shop design system.

## Structure

- One shared CSS file: `styles.css`
- One HTML file per page
- Inline JavaScript only for mobile sidebar, custom selects, and bulk checkboxes
- Fully linked pages through sidebar, tables, action buttons, breadcrumbs, and coverage map
- Responsive design for desktop, tablet, and mobile phones

## Main entry

Open `index.html`.

## Coverage

Open `coverage.html` to see the full route/entity map.

Covered modules:

- Dashboard and analytics
- Products: all, one, form
- Categories: all, one, form
- Brands: all, one, form
- Attributes: all, one, form
- Orders: all, one
- Customers/user profiles: all, one
- Reviews and ratings: all, one
- Carts: all, one
- Checkouts: all, one
- Coupons: all, one, form
- Inventory: all, one
- Admin users: all, one
- Settings

## Notes

This is a static UI prototype. Forms and actions are visual only.
