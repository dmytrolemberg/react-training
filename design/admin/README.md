# Minimal Shop UI Kit

This package contains three connected parts:

1. `shop/` — customer-facing online shop prototype.
2. `admin/` — separate responsive admin portal for managing the shop.
3. `design-guidelines/` — shared design guidelines and component examples.

Open `index.html` first, then navigate to the shop, admin portal, or design guidelines.

## Design direction

The shop and admin portal use the same warm minimal visual language:

- Background: `#F6F5F1`
- Surface: `#FFFFFF`
- Soft surface: `#FAF9F6`
- Border: `#DEDAD0`
- Primary text/action: `#101010`
- Muted text: `#67645F`
- Large rounded cards
- Soft shadows
- Custom checkboxes and selects
- Responsive layouts for desktop, tablet, and mobile

## Main entry points

- `shop/index.html`
- `admin/index.html`
- `design-guidelines/index.html`

## Admin entity coverage

The admin portal includes list/detail/form or management pages for:

- Dashboard
- Products
- Categories
- Brands
- Attributes
- Inventory
- Orders
- Carts
- Checkouts
- Customers
- Reviews and ratings
- Coupons
- Admin users
- Settings

`admin/coverage.html` contains a coverage checklist with links.
